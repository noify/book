# 移动端

## 如何优化h5滚动效果

在safari上 `-webkit-overflow-scrolling : touch;`可以快速实现效果。
如果使用了绝对定位，则需要加上z-index。

```css
-webkit-overflow-scrolling: touch;
position:absolute;
z-index:1;
```
